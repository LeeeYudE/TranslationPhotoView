# TranslcationPhotoView

可上下滑动放回的PhotoView。
与Glide配合可加载网络图片。
尚有瑕疵，大致可用。

注意＊使用的activity需要设置为透明主题＊

Demo参考https://github.com/githubwing/DragPhotoView
